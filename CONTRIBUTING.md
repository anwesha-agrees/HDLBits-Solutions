
# Contributing Guidelines

Thanks for shwoing interest in contributing!

## How to Add a Solution

1. Fork this repository.
2. Add your solution in the correct folder.
3. Name the file after the HDLBits problem (for example, `wire.md` or `dff.md`).
4. Make sure your solution is correct and easy to understand.
5. Open a Pull Request with a clear title.

## Guidelines

* Use clean and readable Verilog.
* Add comments where they help explain the logic.
* Keep explanations short and clear.
* If applicable, include diagrams or waveforms.

## Code Of Conduct

Please be respectful and constructive when contributing.
